# FunctionN (type alias)

**Signature**

```ts
export type FunctionN<A extends Array<unknown>, B> = (...args: A) => B
```

**Example**

```ts
import { FunctionN } from 'fp-ts/lib/function'

export const sum: FunctionN<[number, number], number> = (a, b) => a + b
```

Added in v2.0.0
